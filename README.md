## üåü About the Portfolio
This is **personal portfolio website** showcasing skills, projects, and professional experience.  
The site is **frontend-only**, featuring modern animations, smooth scrolling, and interactive elements.

**Key highlights:**
- Responsive design for mobile, tablet, and desktop.
- Hero section with animated text and parallax background.
- Custom cursor and cursor trail animations.
- Skills section with interactive progress bars.
- Projects showcase with demo and GitHub links.
- Experience timeline with professional history.
- Contact form with social media integration.
